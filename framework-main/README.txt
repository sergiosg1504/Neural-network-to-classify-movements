· Repository architecture:

Framework/
├─ __init__.py
├─ .gitignore
├─ activation_fuctions.py
├─ neuralNetwork.py
├─ normalScaler.py
├─ README.txt
├─ standardScaler.py
├─ utils.py


- __init__.py: file that allows the repository to be included as a package.
- .gitignore: file contaning the files that should be ignored by git. (ignore it)
- activation_fuctions.py: python file that contains all the code about activation functions.
- neuralNetwork.py: python file that contains all the code about Neural networks.
- normalScaler.py: python file that contains all the code for normalization.
- README.txt: file explaning repository architecture.
- standardScaler.py: python file that contains all the code for standarization.
- utils.py: python file that contains some functions that are util for the framework.
